- Does the code correctly implement the intended functionality?
- Are there any logical bugs or edge cases I missed?
- How does it handle invalid inputs or failures?

- Is the code modular and well-organized?

- Is this over-engineered or under-engineered?

- Am I handling hallucinations or bad outputs properly?

- Should I improve retrieval (chunking, ranking, hybrid search)?
